CSS styling effects

Lines, margins, grids, and holes...

I wanted to make some styled elements for writing on based off real-world examples. I wanted to try emulate some types of paper found in text-books and journals such as; ruled, grid, margin, perforated, hole-punched etc. I will showcase some of the styles below.

This section for instance is a combination of a few styles, I wanted it to look like a ruled textbook page with a red margin
I created the lined paper effect using the CSS repeating-linear-gradient() function. By using 'em' units (derived from the parent elements font size) the lines should always adjust to fit the font-size. It's worth noting that differrent font's sizing varies, so some may sit closer to the ruled lines than others.
